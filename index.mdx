## What is Underground Fleet?

Underground Fleet is a web-based vehicle reservation platform built with React. Users can:

- **Browse** a catalog of available vehicles with real-time pricing
- **Reserve** any vehicle by selecting pickup/dropoff dates and location
- **Manage** existing reservations from a personal dashboard
- **Check in online** up to 24 hours before pickup using a reservation code

<Info>
  Underground Fleet connects to a backend REST API at `http://localhost:8080`. All reservation and authentication actions require a running backend instance.
</Info>
