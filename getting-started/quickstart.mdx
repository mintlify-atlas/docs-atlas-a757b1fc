---

This guide walks you through the full flow from installation to completing a reservation and online check-in.

<Note>
  Underground Fleet requires a running backend REST API at `http://localhost:8080`. Start the backend before running the frontend, or authentication and data requests will fail.
</Note>

<Steps>
  <Step title="Prerequisites">
    Before you begin, make sure you have:

    - **Node.js 18 or later** — [nodejs.org](https://nodejs.org)
    - **A running backend API** at `http://localhost:8080` with endpoints for `/auth/login`, `/auth/register`, `/vehicles`, and `/reservations`
  </Step>

  <Step title="Clone the repository and install dependencies">
    Clone the repository and install the required packages.

    <CodeGroup>

    ```bash npm
    git clone https://github.com/Mambee73/reservas-vehiculos.git
    cd reservas-vehiculos
    npm install
    ```

    </CodeGroup>
  </Step>

  <Step title="Start the development server">
    Run the dev server with Vite:

    ```bash
    npm run dev
    ```

    Vite starts the app at `http://localhost:5173`. Open that URL in your browser.

    The available scripts are:

    | Script | Command | Description |
    |---|---|---|
    | `dev` | `npm run dev` | Start local dev server at `http://localhost:5173` |
    | `build` | `npm run build` | Compile and bundle for production |
    | `preview` | `npm run preview` | Preview the production build locally |
    | `lint` | `npm run lint` | Run ESLint across the codebase |
  </Step>

  <Step title="Register an account">
    Navigate to `http://localhost:5173/register` and fill in the three fields:

    - **Usuario** — your username
    - **Correo electrónico** — your email address
    - **Contraseña** — your password

    Submitting the form sends a `POST /auth/register` request. On success, your JWT token is stored in `localStorage` and you are redirected to `/vehicles` automatically — no separate sign-in step needed.

    <Tip>
      Already have an account? Navigate to `http://localhost:5173/login` instead and enter your username and password. A successful login sends `POST /auth/login`, receives a JWT, stores it in `localStorage`, and redirects you to `/vehicles`.
    </Tip>
  </Step>

  <Step title="Browse the vehicle catalog">
    The `/vehicles` page loads automatically after login. It fetches the vehicle list from GET `/vehicles` and displays each vehicle as a card.

    You can also navigate directly to `http://localhost:5173/vehicles` using the **Vehículos** link in the navbar.

    <Tip>
      The home page at `/` includes a booking form with fields for pickup location, pickup date and time, dropoff date and time, and an optional **discount code**. You can enter a discount code here before browsing vehicles to apply it to your reservation.
    </Tip>
  </Step>

  <Step title="Make a reservation">
    On the `/vehicles` page, click the **Reservar** button on any vehicle card. This sends a POST request to `/reservations` with the `vehicleId`, `startDate`, and `endDate`, then redirects you to `/reservations`.
  </Step>

  <Step title="View your reservations">
    Navigate to `http://localhost:5173/reservations` or click **Mis Reservas** in the navbar.

    This page is protected — you must be signed in to access it. It fetches your bookings from GET `/reservations/mine` and displays each one as a card showing the vehicle name, start and end dates, and status. You can cancel any active reservation from this page.
  </Step>

  <Step title="Complete online check-in">
    Navigate to `http://localhost:5173/checkin` or click **Check-in** in the navbar.

    Fill in the three required fields:

    - **Last name** — the last name on the reservation
    - **Reservation code (IRN)** — the unique code from your booking confirmation
    - **Pickup date** — the date your reservation begins

    Submit the form to complete your check-in before arriving for pickup.
  </Step>
</Steps>
